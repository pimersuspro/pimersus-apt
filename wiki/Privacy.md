# Privacy Features

Pimersus Browser includes a set of light privacy protections designed for everyday browsing without breaking website compatibility.

## Included Protections

- Timezone spoofing (UTC)
- Language spoofing (en-US)
- Geolocation disabled
- WebRTC leak prevention
- Clipboard access protection
- Reduced fingerprinting

## Limitations

Pimersus Browser is not an anonymity tool and does not replace Tor or hardened privacy browsers.
